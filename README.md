A basic LLM-based text summarisation project using Ollama and storing the outputs in a sqlite database using SQLModel in Python. 

I have used the two models llama3.1:8b and granite4:tiny-h 
on Catapult's case studies text which are available on Catapult's website. I am testing the right model and SQLModel functionality to add this to my Catapult case study project.
